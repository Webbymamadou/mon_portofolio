## 🚀 À propos du projet

Ce portfolio est une application Web moderne conçue sous forme de **Single Page Application (SPA)**, développée avec des technologies modernes et performantes. Il met en valeur mes compétences académiques et mes réalisations de projets pratiques.

### 🌟 Fonctionnalités clés :
*   **Performance & Rapidité :** Propulsé par **Vite 6** et **React 19** pour un rendu instantané et une fluidité totale.
*   **Micro-animations Premium :** Effets de transition, jauges circulaires en SVG animé et entrées progressives à l'écran conçues avec **Framer Motion**.
*   **Scroll Spy réactif :** Menu de navigation intelligent en React natif avec défilement fluide vers les sections de la page.
*   **Intégration d'API :** Formulaire de contact fonctionnel avec envoi de courriels automatisé via l'API client de **Web3Forms**.
*   **Design responsive :** Interface entièrement adaptative (Mobile, Tablette, PC) reposant sur une grille Sass/Bootstrap v4 personnalisée.

## 💻 Démarrage en local

Pour installer et exécuter ce projet sur votre machine locale, suivez ces étapes simples :

### Prérequis
Assurez-vous d'avoir installé [Node.js](https://nodejs.org/) (version 18 ou supérieure recommandée) sur votre système.

### 1. Cloner le dépôt
```bash
git clone https://github.com/Webbymamadou/mon-portfolio.git
cd mon-portfolio
```

### 2. Installer les dépendances
```bash
npm install
```

### 3. Lancer le serveur de développement local
```bash
npm run dev
```
Ouvrez ensuite votre navigateur sur `http://localhost:5173` pour visualiser le site.

### 4. Compiler pour la production
```bash
npm run build
```
Les fichiers optimisés et prêts pour le déploiement seront générés dans le dossier `/dist`.
